Game Mechanics

modified wooden toy train a.k.a. BRIO.

  • Turn-based game on discrete tracks

  • physics of sss, ttt, vvv as discrete values

  • processes and interaction by the participants an token

  • simulation

  • cooperative game

  • track movement

  • linear tech tree
